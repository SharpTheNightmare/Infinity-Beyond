# Alpha Testing Tools / Mod Menu
<b> Uses MelonLoader (Mono). </b>Precompiled versions are available, or you can build it yourself with visual studio.

Features:

- [X] Client-side FakeDev
- [X] Loaders:
  - [x] Shop
  - [x] Merge
  - [x] Quests (+ Abandon Quests)
- [x] Autoattack + Config
- [x] Packet Tools:
  - [x] Interceptor (halt communications)
  - [x] Sniffer (log Server + Client packets)
  - [x] Sender (send commands to the server)
- [ ] More to come...

Feel free to open pull requests and contribute.
